# Visión del Producto
# Nombre

LJ Agenda

# Objetivo

Ayudar a consultorios independientes a organizar su agenda, reducir las citas perdidas y mejorar la atención mediante un sistema sencillo de reservas.

# Problema

Muchos consultorios pequeños administran sus citas mediante llamadas telefónicas, WhatsApp o agendas físicas.

Esto ocasiona:

- pérdida de pacientes
- cancelaciones
- desorden
- doble reserva
- tiempo perdido

# Solución

Una aplicación web donde los pacientes puedan reservar una cita en cualquier momento y el profesional administre toda su agenda desde un solo lugar.

# Cliente ideal
- Odontólogos
- Psicólogos
- Nutricionistas
- Médicos independientes
- Fisioterapeutas

# MVP

# - Versión 1

- Agendar citas
- Agenda del día
- Pacientes
- Recordatorios
- Confirmación de citas

# - Lo que NO tendrá
- Historia clínica
- Facturación
- Inventario
- Nómina
- Contabilidad
